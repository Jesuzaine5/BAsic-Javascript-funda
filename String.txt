String methods
These differenet methods  can manipulate strings in different ways
toUpperCase() --  word. to Uppercase();
toLowerCase() --  word. to Lowerrcase();
trim()-- word.trim(); -removes extra spaces on start and end of a string
trimEnd()--  word.trimEnd();
trimStart()--  word.trimStart();
replace()-- word.replace(dog,cat); replaces first matching word on a string
replaceAll()-- word.replaceAll(dog,cat);-- replace all matching word on a string
slice()-- word.slice(start, ending); -create a part of a string by specifying the start and end index

create string values by using ``(backticks)
let word = `String "Literal"`;
